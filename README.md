# Packaging Tool

一个还在长身体的包装工程网页小工具。  
目前主要用于**公司内部测试、逻辑验证、规则演练**，不是正式产品，也不是随便拿走就能上岗的万能神器。

---

## 这是什么

这是一个用于包装工程参数化出图的测试项目，当前主要功能包括：

- 输入基础尺寸参数
- 自动计算部分结构参数
- 生成刀版平面图
- 生成简化 3D 斜侧示意图
- 导出 SVG
- 持续尝试补充 PDF 等导出能力

简单说，它现在更像是：

> 一个认真工作的实验体，而不是一个已经西装革履去开发布会的正式产品。

---

## 项目性质

**这是一个公司测试项目。**

请特别注意：

- 本项目仅用于公司内部测试、演示、验证和开发讨论
- 当前仍处于原型阶段，规则、界面、导出逻辑都可能随时调整
- 本项目**不是正式发布工具**
- 本项目**不是对外产品**
- 本项目**不是下载回去就能放心投入生产的成熟系统**

---

## 当前功能

目前已实现：

- 长度 / 宽度 / 高度 / 厚度输入
- 自动计算派生参数
  - Gap
  - Glue Flap
  - Flipcover
  - Glue Flap Height
- 三栏界面布局
  - 左侧：参数输入与计算值
  - 中间：刀版平面图
  - 右侧：3D 斜侧示意图
- SVG 导出
- 多文件结构拆分，便于继续完善
